One-time table setup (SQL)
Run this once in the Supabase SQL editor:
create table if not exists navigator_entries (
  id text primary key,
  sync_code text not null,
  week_of text,
  data jsonb,
  updated_at timestamptz default now()
);
alter table navigator_entries enable row level security;
create policy "navigator open" on navigator_entries
  for all using (true) with check (true);

Passphrase

Changing your passphrase re-encrypts every entry with the new one. There is no recovery if it's forgotten — that's what keeps the journal private.
